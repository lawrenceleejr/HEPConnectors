# Connectors of the Particle Physics Lab

A print-ready wall poster / field guide to the connectors you meet at the crate,
the rack and the counting room — coax, fibre, network, data, video, audio and
power. Male / female is shown wherever the gender differs, each entry has a
one-line "what it's for," and **colour encodes the signal medium**: copper =
electrical, aqua = optical fibre (which is also the real OM3/OM4 jacket colour).

| File | What it is |
|------|------------|
| [`poster.html`](poster.html) | The poster — a single self-contained HTML file (fonts inlined, no network needed). Open in any browser; adapts to light/dark. |
| `poster-light.png` | High-resolution rendered poster, light "datasheet" theme. |
| `poster-dark.png` | High-resolution rendered poster, dark "control-room / scope" theme. |
| `build/` | Reproducible source: template, font files, build + render scripts. |

## What's on it

- **RF & coaxial** — BNC, SMA, SMB/SMC, MCX/MMCX, N-type, TNC, LEMO, Twin-BNC / Twinax
- **Fibre optic** — LC, LC-LC duplex, SC, ST, FC, MTP/MPO, fusion splice, MU
- **Network & transceivers** — RJ45, RJ11, SFP/SFP+, QSFP+/QSFP28, DAC / twinax
- **Data, serial & instrumentation** — USB (A/B/Micro/Mini/C), DB9, DB25, HD-15/VGA, GPIB (IEEE-488), FMC (VITA 57), IDC ribbon
- **Video & display** — HDMI (A/C/D), DisplayPort, DVI
- **Audio, analogue I/O & power** — RCA, 3.5/6.35 mm phone jack, TOSLINK, XLR, banana, DC barrel, IEC C13/C14, Molex
- **Reference** — fibre-by-jacket-colour chart (OM1–OM5, OS1/OS2), UPC vs APC polish, and field cheats (M vs F, 50 vs 75 Ω, SM ≠ MM)

## Printing

`poster.html` prints cleanly from any browser; the PNGs are rendered at high DPI
(~3900 px wide) and suit large-format printing at **A1** or **24 × 36 in**. The
proportions are those of a tall wall chart — a print shop can scale to fit.

## Rebuilding

The poster is generated by inlining the six woff2 fonts into the template:

```bash
cd build
python3 build.py ../poster.html          # writes the self-contained poster.html

# optional: re-render the preview PNGs (needs Node + Playwright/Chromium)
node render.mjs ../poster.html ../poster-light.png light 2.6
node render.mjs ../poster.html ../poster-dark.png  dark  2.6
```

Type: **Archivo** (display), **IBM Plex Sans** (body), **IBM Plex Mono** (labels)
— all Open Font License, embedded directly in the HTML.
